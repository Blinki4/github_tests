# GitHub автотесты

Фреймворк для тестирования GitHub, основанный на Pytest, Selenium и Requests

## Запуск тестов

Для того, чтобы успешно запустить тесты, необходимо:
1) Склонировать репозиторий
2) Создать в корне проекта файл .env
4) Определить внутри файла переменные:
   1) API_KEY='ВАШ API КЛЮЧ ОТ GITHUB'
   2) LOGIN='ВАЛИДНЫЙ ЛОГИН ОТ GITHUB'
   3) PASSWORD='ВАЛИДНЫЙ ПАРОЛЬ ОТ GITHUB'
   4) REPO_NAME='НАЗВАНИЕ ДЛЯ ТЕСТОВОГО РЕПОЗИТОРИЯ'
5) Выполнить команду pytest -v
   - Запуск только UI тестов: pytest -v -m ui
   - Запуск только API тестов: pytest -v -m api
