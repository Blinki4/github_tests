# GitHub автотесты

Фреймворк для тестирования GitHub, основанный на Pytest, Selenium и Requests

## Запуск тестов

Для того, чтобы успешно запустить тесты, необходимо:
1) Склонировать репозиторий
2) Создать в корне проекта директори test_data
3) Внутри test_data создать файл credentials.py
4) Определить внутри файла переменные:
   1) API_KEY = 'ВАШ API КЛЮЧ ОТ GITHUB'
   2) valid_login = 'ВАЛИДНЫЙ ЛОГИН ОТ GITHUB'
   3) valid_password = 'ВАЛИДНЫЙ ПАРОЛЬ ОТ GITHUB'
   4) new_repo_name = 'НАЗВАНИЕ ДЛЯ ТЕСТОВОГО РЕПОЗИТОРИЯ'
5) Выполнить команду pytest -v
   1) Запуск только UI тестов: pytest -v -m ui
   2) Запуск только API тестов: pytest -v -m api
